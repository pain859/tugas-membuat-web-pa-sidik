Command Prompt (CMD)

Instruksi baris perintah dasar yang wajib dikuasai oleh siswa TJKT untuk troubleshooting jaringan:


Administrator: Command Prompt

C:\Users\Palguna> ipconfig // Cek konfigurasi IP

C:\Users\Palguna> ping 8.8.8.8 // Tes koneksi internet

C:\Users\Palguna> tracert google.com // Lacak rute jaringan

C:\Users\Palguna> nslookup // Cek DNS Server

C:\Users\Palguna> netsh wlan show profiles // Cek profil WiFi

IP Address & Subnetting

IPv4

Alamat 32-bit yang dibagi menjadi 4 oktet. Digunakan untuk identitas perangkat di jaringan lokal maupun global.

Subnetting

Proses membagi jaringan besar menjadi sub-jaringan yang lebih kecil (Subnet) untuk efisiensi dan keamanan.

Voltase & Watt Perangkat

Sebagai teknisi, kita harus menghitung beban daya agar perangkat tidak rusak (overload).

Voltase (V): Tekanan listrik. Di Indonesia standar PLN adalah 220V.

Watt (W): Daya nyata yang digunakan perangkat. Rumus: P = V x I (Watt = Volt x Ampere).

Contoh: Laptop butuh 19V dengan arus 3.42A, maka dayanya ±65 Watt.

Aplikasi Virtualisasi

Digunakan untuk simulasi OS atau server tanpa memerlukan hardware tambahan:

  • VirtualBox: Gratis dan paling umum digunakan sekolah.
  • VMware Workstation: Standar industri profesional.
  • Cisco Packet Tracer: Virtualisasi simulasi jaringan Cisco.
